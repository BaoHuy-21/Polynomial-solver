# Polynomial-solver
A C++ program for polynomial arithmetic, featuring long division and automatic simplification algorithms.
## Key Features
- **Auto-Simplification:** Merges terms with the same exponent and sorts them in descending order.
- **Full Arithmetics:** Addition, Subtration, Multiplication.
- **Polynomial Long Division:** Handles complex division with quotient and remainder.
- **Calculus:** First-order derivative calculation.
- **Evaluation:** Calculates the polynomial value for a given x.
